# FAQ 知识库查询技能

## 强制执行规则 — 每次收到用户问题都必须执行

**你必须使用 exec 工具执行以下命令来读取知识库文件，然后才能回答用户。
禁止跳过这一步，禁止凭自己的知识直接回答。**

### 第一步：选择要读取的文件，然后执行命令

用户问题涉及 **账号、注册、密码、项目、上传、个人信息、团队、技术支持**：
```
cat knowledge_base/general.md
```

用户问题涉及 **收费、套餐、发票、退款、API 使用量、付费、订阅、账单**：
```
cat knowledge_base/billing.md
```

如果不确定属于哪类，两个都读：
```
cat knowledge_base/general.md knowledge_base/billing.md
```

### 第二步：在文件内容中查找匹配的问答

知识库文件格式为：
```
## Q: 问题标题
答案内容...
```

找到和用户问题最匹配的 Q 条目，提取其下方的答案内容。

### 第三步：按知识库内容回答

- 找到匹配 → 引用知识库中的步骤/信息回答用户
- 找不到匹配 → 回答"知识库中没有找到相关信息，是否需要我帮您创建工单转交人工处理？"

### 绝对不能做的事

- ❌ 不读文件直接回答
- ❌ 凭自己训练数据中的知识回答
- ❌ 编造知识库里没有的信息
