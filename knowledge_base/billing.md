# 平台收费

平台采用订阅制收费模式，按团队规模和功能模块计费。

目前提供以下套餐：

* Free 免费版
* Pro 专业版
* Enterprise 企业版

用户可按月或按年订阅。

---

# 免费版限制

免费版包含：

* 最多 3 名团队成员
* 每月 1000 次 API 调用
* 5GB 文件存储空间

超出额度后需要升级套餐。

---

# 升级到专业版

操作步骤：

1. 登录系统
2. 进入「账单与订阅」
3. 点击「升级套餐」
4. 选择 Professional Plan
5. 完成支付

升级后立即生效。

---

# 年付和月付区别

月付：

* 按月扣费
* 灵活取消

年付：

* 一次支付 12 个月费用
* 可享受 20% 折扣

---

# API 调用次数计算

以下行为会消耗 API 配额：

* 调用聊天接口
* 调用知识库检索接口
* 调用文档分析接口

系统任务和后台同步任务不计入配额。

---

# 查看 API 使用量

查看步骤：

1. 进入「控制台」
2. 点击「使用统计」
3. 查看 API 调用情况

系统会展示：

* 已使用额度
* 剩余额度
* 调用趋势图

---

# 超出套餐额度怎么办

当达到套餐限制时：

* 系统发送邮件提醒
* API 返回额度不足提示
* 用户可升级套餐继续使用

系统不会自动扣费。

---

# 企业版定制报价

支持企业版定制报价。

企业版客户可根据以下因素获得专属报价：

* 团队人数
* API 调用量
* 数据存储需求
* SLA 服务等级

---

# 退款政策

购买后 7 天内且未大量使用服务的情况下，可申请退款。

退款申请需提交工单审核。

---

# 下载发票

进入：

账单与订阅 → 发票管理

即可下载电子发票。

企业客户可申请增值税专用发票。
